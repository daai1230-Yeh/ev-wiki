# BMW（寶馬）

德國豪華汽車製造商，全球最大豪華車集團（BMW、MINI、Rolls-Royce 三品牌）。2026 年以 iX5 世界首映為標誌，進入高續航旗艦電動車市場，同時推進 Neue Klasse 全新電動平台架構。

## 基本資訊

| 項目 | 細節 |
|------|------|
| 總部 | 德國慕尼黑 |
| 主要品牌 | BMW、MINI、Rolls-Royce |
| 電動車平台 | **Neue Klasse**（全新 BEV 專用平台，2025+ 量產）|
| 電池合作夥伴 | LGES（46 系列圓柱，100 GWh 合約）、Samsung SDI（匈牙利）|

## BMW iX5 世界首映（2026-06-30）

BMW 旗艦電動 SUV 全球首映，進入新能量密度區間：
| 規格 | 數值 |
|------|------|
| 車型 | **BMW iX5**（旗艦電動 SUV）|
| 電池容量 | **141 kWh** |
| 最大功率 | **425 kW（579 hp）** |
| 最大扭矩 | **805 Nm** |
| 0-100 km/h | **4.6 秒** |
| WLTP 續航 | **645–845 km** |
| DC 快充峰值 | **460 kW** |
| AC 充電 | **22 kW** |
| 10–80% 時間 | **23 分鐘**（460 kW 峰值）|
| 拖曳能力 | **2,700 kg** |
| 生產開始 | **2026-08**（南卡羅萊納州 Spartanburg 廠）|
| 市場上市 | **2027 年初** |
| 氫能版 | **2028 年**（燃料電池版本）|

- Spartanburg 廠：BMW 全球最大製造廠，生產所有 X 系列車型
- iX5 氫能版：BMW 堅持氫燃料電池乘用車路線（全球極少數仍在量產 FCV 乘用車的廠商）

## ADAS 系統

| 功能 | 描述 |
|------|------|
| **Entry-2-Exit** | 停車場自動駕駛（進入→停車→離開全程自動）|
| **Address-2-Address** | 點對點 L2 輔助駕駛（高速公路 + 城市）|

- BMW 的 ADAS 策略：強調「系統安全可靠」而非宣傳等級，L2+ 在歐洲市場定位謹慎

## 電池供應策略

### LGES × BMW：100 GWh 46 系列圓柱大單（2026-04-30）
| 格式 | 應用 | 合約規模 |
|------|------|---------|
| **4695**（46mm × 95mm）| BMW iX3/i3/i7（Neue Klasse）| ~10 兆韓元（€58 億），10 年 |
| **46120**（46mm × 120mm）| BMW iX5 等大型車 | 同合約範圍內 |

- LGES 亞利桑那廠 2026 年底完工，地理位置對應 BMW Spartanburg 廠
- 合約為 LGES 首次正式進入 BMW BEV 供應鏈（過去僅供 BMW 混動）

### Samsung SDI
- BMW 長期 NMC 供應商；匈牙利廠為歐洲主要供應節點
- 可能引入 Samsung SDI 額外 2 條方形電池產線（強化 EU 本土化）

## Neue Klasse 平台架構

| 特性 | 細節 |
|------|------|
| 架構 | 全新 BEV 專用（非改造油電平台）|
| 電池整合 | 6th gen 電池，能量密度 +20% vs 現行 |
| 充電速度 | 顯著提升（Neue Klasse 首款 BMW i3：108 kWh，900 km，400 kW）|
| 目標市場 | 全球，包含中國、美國、歐洲 |

- BMW CLAR 平台：中國市場 iX1/i3/i5（CLAR 平台）2026-07 起廢除，Neue Klasse 接棒
