# Zenobē £9.8 億融資：再支持 1,200 輛電動巴士（2026-06-17）

**原文**：electrive.com｜**發布**：2026-06-17

## 融資規模

| 輪次 | 金額 |
|------|------|
| 本輪（第三輪）| **£9.8 億** |
| 累計融資總額 | **>£32 億**（自 2017 年） |
| 歐洲 EV 車隊融資設施（2025）| €3.25 億 |

## 影響範圍

- **1,200 輛**以上電動巴士（含充電基礎設施）
- 時程：未來 3 年
- 2028 年前支持英國+歐洲 **5,000 輛**電動巴士

## Zenobē 現況

| 指標 | 數值 |
|------|------|
| 支持電動車輛 | 3,400+ 輛（120 個停車場）|
| 英國電動巴士市占 | ~**25%** |
| 新市場 | 澳洲（£1 億電動卡車基金）、北美（收購 Revolv）|

## 商業模式

Zenobē 提供**捆綁服務**：車輛融資 + 充電基礎設施 + 車隊電動化顧問，讓運營商無需分別管理各環節。

核心財務論點：柴油巴士燃料佔全生命週期成本 ~40%；電動巴士電費佔 ~20%。
