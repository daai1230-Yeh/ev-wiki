# 本田 × QuantumScape 固態電池合作（2026-06-19）

**原文**：InsideEVs｜**發布**：2026-06-19

## 合作概況

本田（Honda）與固態電池新創 QuantumScape 簽署**聯合研究協議（Joint Research Agreement）**，聚焦固態電池開發及製造流程研究，為多年期計畫。

## 背景

| 要點 | 說明 |
|------|------|
| 本田近況 | 剛取消多款原本預計 2026 年美國上市的次世代 EV 計畫 |
| 合作意涵 | 儘管短期 EV 計畫縮水，本田仍視固態電池為長期核心機會 |
| QuantumScape | 美國固態電池新創，VW 集團為主要投資方；2026 年初啟用 Eagle Line 自動化產線（Cobra 製程）|

## QuantumScape 技術現況

- 採用鋰金屬固態技術，開發逾十年
- Eagle Line：高度自動化產線，代表量產藍圖
- 已有 VW 集團深度投資關係

## 本田自研固態電池目標

| 指標 | 目標 |
|------|------|
| 體積 | 比現有鋰離子電池小 **50%** |
| 重量 | 輕 **35%** |
| 成本 | 低 **25%** |
| 續航 | 超過 **1,000 km**（620 英里，未來可達 700+ 英里）|
| 量產時間 | 2020 年代後半（示範生產線已運行）|

## 意義

- 本田同時推進自研路線 + 外部合作，採多元布局
- QuantumScape 原為 VW 陣營；此合作跨越 OEM 陣營邊界
- 參見固態電池競賽：日產（23 層堆疊驗證）、Toyota、Solid Power（BMW/Ford）、Honda 自研
