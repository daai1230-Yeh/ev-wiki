# Rivian × ChargeScape：離峰自動充電 + R2 V2G 支援（2026-06-16）

**原文**：InsideEVs｜**發布**：2026-06-16

## 摘要

Rivian 與 ChargeScape（BMW、Ford、Honda、Nissan 投資）簽署合作，讓 R1S、R1T、R2 車主自動在離峰時段充電節省費用，並為 R2 加入 **V2G（vehicle-to-grid）** 賣電返電網功能（OTA 更新）。

## 關鍵事實

| 項目 | 內容 |
|------|------|
| 適用車型 | R1S、R1T（現有）；R2（V2G via OTA）|
| 功能 | 自動離峰充電（降低充電成本） |
| V2G | R2 支援：低電價充電、高電價返電 |
| ChargeScape 支持公用事業 | PG&E、TXU Energy、Xcel、Con Edison、DTE |
| ChargeScape 離峰充電率 | 89% EV 充電在離峰時段完成 |

## ChargeScape 背景

- 德州 vehicle-grid integration 平台
- 股東：BMW、Ford、Honda、Nissan
- 其他用戶：Tesla、Stellantis

## 意義

- Rivian R2 成為繼 Ford F-150 Lightning 之後，美系主流品牌支援 V2G 的又一案例
- 充電成本降低 → 提升 EV 吸引力
- 電網穩定化 → EV 車隊成為分散式儲能資產
