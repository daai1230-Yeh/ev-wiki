# VW ID. Buzz 2026 款年改：新增 AWD、One-Pedal、V2L（2026-04-22）

**原文**：electrive.com｜**發布**：2026-04-22

## 核心更新

| 功能 | 規格 |
|------|------|
| 新 AWD 版本 | ID. Buzz **Pro 4MOTION**（採用 GTX 動力系統） |
| 系統功率 | **250 kW**（後軸 210 kW + 前軸附加電機） |
| 電池（標準軸距）| 淨 **79 kWh** |
| 電池（長軸距 +25cm）| 淨 **86 kWh** |
| 新功能 | **單踏板駕駛（One-Pedal）+ V2L（Vehicle-to-Load）** |

## 背景

- ID. Buzz Pro 4MOTION 使用與 GTX 相同的驅動系統，兩者系統功率均為 250 kW
- V2L 功能擴展 ID. Buzz 商用場景（電源輸出、營地等）
- 更新預定 2026 年夏季上市

## 意義

VW 在 ID. Buzz 年改中加入 V2L，跟進 Hyundai、Kia 等競品的標準配備，顯示 V2G/V2L 功能正從高端選配向主流滲透。
