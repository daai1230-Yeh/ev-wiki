# Waymo

Alphabet（Google 母公司）旗下自動駕駛公司，2026 年為全球最大商業 Robotaxi 營運商，正從美國向全球擴張。

## 現況（2026-06）

| 指標 | 數值 |
|------|------|
| 車隊規模 | **近 4,000 輛** |
| 服務城市 | **10+ 個**（SF、LA、Phoenix、Austin、Atlanta 等）|
| 覆蓋面積 | **>1,400 平方英里** |
| 乘車量 | **~50 萬次/週** |
| 2026 年目標 | **100 萬次付費乘車/週** |
| 海外擴張 | **東京** + **倫敦**（2026 下半年）|
| 近期融資 | $160 億（估值 $1,260 億）|
| 累計自動里程 | **>2 億英里** |

## Apple Titan 測試場收購（$2.2 億，2026-06）

Waymo 以 **$2.2 億美元**收購 Apple 位於亞利桑那州的 **5,500 英畝**自駕車測試場（Project Titan 舊址）：

| 設施 | 規模 |
|------|------|
| 城市道路模擬區 | 115 英畝 |
| 車輛動態測試區 | 35 英畝 |
| 橢圓測試車道 | 4 英里 |
| 高速公路測試段 | 有 |

- Apple 2021 年以 $1.25 億買入 → 2024 年放棄 Project Titan → Waymo 以 1.76 倍溢價接手
- 5,500 英畝專用測試場是競爭對手難以複製的基礎設施優勢
