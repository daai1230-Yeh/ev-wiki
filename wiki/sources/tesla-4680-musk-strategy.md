# Tesla 4680 電池技術 + Musk 垂直整合戰略

**原文**：DIGITIMES／黃女瑛、林廷宇｜**發布**：2026-04-10

## Tesla 4680 電池技術概覽

| 規格 | 數值 |
|------|------|
| 尺寸 | 直徑 46mm × 高度 80mm |
| 設計亮點 | 無極耳（tabless）設計，縮短電流路徑 |
| 電極製程 | 乾式電極（2026 年實現負極+正極同時穩定量產）|
| 現行應用 | Cybertruck、部分 Model Y |
| 供應商 | Tesla 自建 + Panasonic |

### 技術演進
- 2020 年發布，初期乾式電極製程良率是瓶頸
- 2026 年實現穩定量產，持續開發新正極化學配方（NC-19、NC-Mineral）
- 寧德時代等廠商也在跟進開發類似大尺寸圓柱電池

## Musk 垂直整合戰略

### 核心邏輯：掌控「技術密碼」
- 掌握核心技術與成本結構 → 議價優勢 + 推動自製能力
- 電芯依賴 Panasonic / LGES / CATL，但 BMS 與三電系統核心控制權自持

### 跨產業布局
| 業務 | 說明 |
|------|------|
| Tesla（EV）| 4680 電池、FSD、Robotaxi |
| SpaceX（衛星/太空）| Starlink 衛星用太陽能 + 電池（關鍵封裝由 Tesla 掌握）|
| Optimus（機器人）| 垂直整合延伸至人形機器人 |
| Terafab（半導體）| 德州晶圓廠計畫，延伸製造能力 |

### 美國製造策略
- 密集接觸中國 HJT/BC/鈣鈦礦技術供應商 → 吸收並移植至美國
- 目標：建立年產能 100 GW 美國太陽能製造體系
