# Mobileye 自建 Robotaxi 業務：2027 美國上路，17,000 輛 5 年目標（2026-06-18）

**原文**：electrive.com + Digitimes｜**發布**：2026-06-17~18

## 核心公告

Mobileye 宣布成立**垂直整合 Robotaxi 部門**，自行開發並營運無人叫車服務（而非僅提供技術）。

## 時程規劃

| 階段 | 內容 |
|------|------|
| **2027 年** | 美國某主要城市上線，初始 **100 輛** |
| 2027 全年 | 分階段推進 |
| **5 年內** | 擴展至 **17,000 輛** |

## 技術架構

- **Mobileye Drive**（自動駕駛系統）
- **Moovit**（子公司）：城市移動數據、行程規劃、3,500+ 城市、11.2 億用戶
- **不自製車輛**：與外部車輛平台製造商和車隊整合夥伴合作

## 市場競爭格局（2026 年）

| 業者 | 狀況 |
|------|------|
| **Waymo** | 領先：11 個城市，$29.99/月訂閱；2026 年進軍倫敦/東京 |
| **Zoox（Amazon）** | 預計 2026 夏透過 Uber App 拉斯維加斯上線 |
| **Tesla** | 德州 ~50 輛（Waymo 規模 10%） |
| **Mobileye** | 2027 起自行進場，Lyft 達拉斯 2026 採用其技術 |

## 複雜性

Mobileye 同時是技術供應商（VW ID.Buzz AD/MOIA、Holon、Verne）又成為競爭者，分析師認為邊界管理（數據/客戶利益）是關鍵挑戰。
