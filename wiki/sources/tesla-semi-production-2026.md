# Tesla Semi 量產啟動（2026-04-30）

**原文**：Electrive（Cora Werwitzke）｜**發布**：2026-04-30

## 摘要
Tesla 電動大卡車 Semi 正式開始高產量量產，首輛從 Gigafactory Nevada 全新生產線下線。這是 Tesla 史上開發周期最長的車型（2017 年亮相，原定 2019 年上路，2026 年才量產）。

## 關鍵數據

| 規格 | 數值 |
|------|------|
| 生產地點 | Gigafactory Nevada 旁獨立廠房 |
| 年產能規劃 | **最高 50,000 輛**（逐步爬坡）|
| 標準版 | 325 英里（~523km），GVW 82,000 磅（~37.2 噸），**$260,000** |
| 長程版 | 500 英里（~800km），**$290,000** |
| 動力 | 三電機整合，最高 **800 kW** |
| 充電 | 支援 **兆瓦充電（1.2 MW）** |

- 售價為美國 Class 8 BEV 半掛車中**最低**
- 預生產車輛：約 36 輛在 PepsiCo 加州車隊運行

## 充電基礎建設

- 首座 Megacharger 站：加州安大略（Ontario, CA）已啟用
- 全美計畫：再增 66 個站點，分布 15 個州

## 市場需求驗證

加州清潔卡車補貼計畫（2025-01 至 2026-02）：
- 共 1,067 份申請，**965 份指定 Tesla Semi**
- 戴姆勒 + Paccar + Volvo 合計 < 100 份
→ **市場需求高度集中於 Tesla Semi**

## 電池規格正式確認（CARB，2026-05-11）

California Air Resources Board（CARB）官方文件正式公布 Tesla Semi 電池容量：

| 版本 | 電池容量（可用）| 動力輸出 | 重量（空車）|
|------|--------------|---------|-----------|
| 標準版 | **548 kWh** | **525 kW**（CARB 資料）| <9.07 噸 |
| 長程版 | **822 kWh** | **800 kW** | 10.43 噸 |

- 電池化學：Tesla 自製 **4680 NCMA** 電芯
- 推測：長程版採 **3 組電池包**，標準版 **2 組**
- 長程版並使用延伸底盤（extended chassis）
- 注意：特斯拉官網仍標示兩版本均 800 kW，但 CARB 文件顯示標準版僅 525 kW
- 理論最大續航計算（822 kWh / 1.7 kWh/mile）= **483 英里**，略低於標榜 500 英里

## 4680 Cybercell（第二代）規格（2026-05-13）

Digitimes 引述 Tesla 官方資料，Tesla Semi 採用的 4680 電芯為「**Cybercell**」（4680 二代），技術規格顯著升級：

| 規格 | Cybercell 數值 |
|------|--------------|
| 能量密度 | **272 Wh/kg**（舊款 4680 約 244 Wh/kg，提升 ~11%）|
| 製造工藝 | **乾電極（dry electrode）**製程 |
| 結構 | **無極耳（tabless）+ 直接焊蓋（direct lid welding）** |
| 充電峰值 | **1.2 MW Megacharger** 峰值充電功率 |

- 乾電極製程為 Tesla 長期攻克的技術難關，可降低製造成本並提升生產速度
- 無極耳設計延續 4680 初代設計，直接焊蓋為新增改良
- 272 Wh/kg 使 Semi 在電池重量相同下可提供更高有效負載餘裕
