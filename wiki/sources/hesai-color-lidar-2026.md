# 禾賽科技 EXT：全球首款顏色辨識 LiDAR（2026-04-20）

**原文**：Digitimes（李佶璋）/ Reuters｜**發布**：2026-04-20

## 摘要
中國 LiDAR 大廠禾賽科技（Hesai）發布 EXT 感測器，搭載自研「畢卡索（Picasso）」晶片，整合空間測距與顏色偵測，為全球首款能辨識顏色的 LiDAR，預計 2027 年搭載旗艦車型。

## 技術規格

| 項目 | 細節 |
|------|------|
| 型號 | **EXT** |
| 核心晶片 | **Picasso**（禾賽自研）|
| 技術突破 | 整合空間測距 + **顏色偵測**（全球首款）|
| 主要應用 | 紅綠燈顏色辨識 → 提升自駕精確度與安全性 |
| 量產時程 | **2027 年**（搭載旗艦車型）|

## 禾賽市場地位（2025）

| 指標 | 數值 |
|------|------|
| 中國市場份額 | **>40%** |
| 主要客戶 | 理想汽車、小米、比亞迪 |
| 國際客戶 | **NVIDIA**（ADAS 核心供應商）|

## 策略意義

- **從成本競爭轉向價值創新**：禾賽 CEO 李一帆：「若只追求低價，將犧牲創造更高價值的創新機會」
- LiDAR vs 純視覺爭議：Tesla / 小鵬偏好純視覺 AI；禾賽認為 LiDAR 仍有成長空間
- **延伸應用**：Kosmo 手持設備 → 人形機器人 + 具身智慧（Embodied AI）
