# Waymo×B2U 電池二次利用 + 科羅拉多州電池回收立法（2026-06-05/08）

**原文**：InsideEVs Critical Materials｜**發布**：2026-06-05

## 摘要
Waymo 與電池二次利用新創 B2U Storage Solutions 合作，將退役的 I-Pace Robotaxi 電池組改裝為**定置儲能系統**；同期美國科羅拉多州成為**首個立法強制電池廢棄管理**的州，要求 2029 年起禁止電池填埋，並在 2035 年前達到 80% 鋰回收率。

## Waymo × B2U 電池二次利用

| 指標 | 內容 |
|------|------|
| 原始電池來源 | Waymo Robotaxi 車隊退役 I-Pace（Jaguar） |
| 再利用形式 | 定置儲能（Stationary Storage） |
| 合作夥伴 | B2U Storage Solutions |
| 預計規模 | 初期批次：數百 kWh（具體未揭露） |
| 商業意義 | 降低電池全生命週期碳排放 + Waymo 電池退役成本 |

### 電動車電池二次利用邏輯
- EV 電池在容量降至原始的 70-80% 後「退役」，但仍可作儲能用途使用 5-10 年
- Robotaxi 電隊車輛高強度使用，退役周期 4-6 年（vs 一般私家車 10-15 年）
- B2U 已在加州建有約 25 MWh 二次利用儲能系統

## 科羅拉多州電池回收立法

| 條款 | 內容 |
|------|------|
| 施行州 | 科羅拉多（Colorado）— 美國**首州** |
| 禁止填埋 | **2029 年起** |
| 回收率目標 | 2035 年 **80% 鋰回收率** |
| 責任主體 | 電池生產商（生產者責任制 EPR） |

### 立法意義
- 美國聯邦層面電池回收法規仍付之闕如
- 科羅拉多帶頭，預計帶動其他州效仿（加州、紐約等在觀察）
- 歐盟《電池法》（Battery Regulation）2030 年回收目標 50% 以上，科羅拉多 80% 目標更嚴
