# Lucid Cosmos：~$50,000 大眾市場 EV，專利圖曝光（2026-06-16）

**原文**：InsideEVs｜**發布**：2026-06-16

## 摘要

Lucid 旗下首款大眾市場車型 **Cosmos** 的專利設計圖曝光，起售價約 $50,000，年底前在沙烏地阿拉伯量產，對標特斯拉 Model Y 與 Rivian R2，並同時作為未來 Robotaxi 平台基礎。

## 關鍵規格

| 項目 | 數值 |
|------|------|
| 起售價 | **~$50,000** |
| 風阻係數 | **0.22** |
| 預估續航 | **>300 英里** |
| 充電架構 | **800V**，14 分鐘補充 200 英里 |
| 雙向充電 | **V2H + V2G 支援** |
| 內裝 | 單一橫跨儀表板大螢幕 |

## 設計特點

- 縮小版 Lucid Gravity 設計語言，Cab-forward 造型
- 後窗大角度傾斜，需數位後視鏡
- 後方「LUCID」大字字母拼排
- 車頂尾翼橫切後窗玻璃

## 量產計畫

- 2026 年底：沙烏地阿拉伯工廠率先量產
- 6~12 個月後：美國亞利桑那州工廠投產

## 戰略意義

- 對 Lucid 類似 Model 3/Y 之於特斯拉的地位——進入主流市場的關鍵
- Cosmos 平台將延伸為 Uber 合作 Robotaxi 車型
- 連結 Lucid 億美元 Uber 夥伴關係
