# ABB M-Series：200kW~1.2MW 分散式充電系統（2026-04）

**原文**：electrive.com／Cora Werwitzke｜**發布**：2026-04-24

## 摘要
ABB E-mobility 推出 M-Series 分散式充電系統，以中央電源機箱搭配衛星充電器（Solo/Duo/Dock/Ultra）的模組架構，提供 200kW 至 1.2MW 的可擴展容量，支援 CCS1/CCS2/NACS/MCS 四種規格。

## 產品架構

```
Power Cabinet（中央機箱）
    ├── Solo Dispenser（單槍）
    ├── Duo Dispenser（雙槍）
    ├── Dock Dispenser（停車型）
    └── Ultra Dispenser（重卡 MCS）
```

## 關鍵規格

| 項目 | 數值 |
|------|------|
| 容量範圍 | **200kW ~ 1.2MW** |
| 支援充電規格 | CCS1 / CCS2 / NACS / **MCS**（兆瓦充電，重卡）|
| 場地配置種類 | **36 種** |
| 電力電子技術 | SiC（碳化矽）半導體，IP54，氣冷 |
| 效率 | ~97%（SiC 優勢）|
| 商業部署時間 | 2024 年起 |

## 設計理念

- **分散式架構**：動態分配電力至充電點，依實際需求即時調整
- **可彈性擴充**：從小型生態系起步，按需追加分配器和電源單元
- **「交付能量經濟學」**：ABB 強調業者轉向以「全生命週期交付能量成本」為考量，而非峰值功率

## 應用場景

- 公共快充站
- 車隊停車場
- 零售場所（參見：Walmart 部署）
- 目的地充電
