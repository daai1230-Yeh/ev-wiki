# Waymo 收購蘋果亞利桑那測試場：$2.2 億擴大 Robotaxi 基礎設施

**事件日期**：2026-06-12｜**金額**：$2.2 億美元

## 核心事件

Alphabet 旗下 Waymo 以 **$2.2 億美元**收購蘋果（Apple）位於**亞利桑那州的 5,500 英畝自駕車測試場**，大幅擴充研發與測試能力。

## 測試場規格

| 設施 | 規模 |
|------|------|
| 總面積 | **5,500 英畝**（遠超 Waymo 加州/俄亥俄州其他基地）|
| 城市道路模擬區 | **115 英畝** |
| 車輛動態測試區 | **35 英畝** |
| 橢圓形測試車道 | **4 英里** |
| 高速公路測試段 | 專用一條 |

## 歷史沿革

| 時間 | 事件 |
|------|------|
| 原建設方 | 飛雅特克萊斯勒（高溫零件測試）|
| 蘋果介入 | Project Titan 自駕計畫租用多年 |
| 2021 | 蘋果以 **$1.25 億**買入 |
| 2024 初 | 蘋果宣布終止 Project Titan |
| **2026** | Waymo 以 **$2.2 億**收購（近買入價 1.76 倍）|

## Waymo 當前擴張現況（2026-06）

| 指標 | 數值 |
|------|------|
| 車隊規模 | **近 4,000 輛** |
| 美國服務城市 | **10+**（SF、LA、Phoenix、Austin、Atlanta 等）|
| 覆蓋面積 | **>1,400 平方英里** |
| 2026 年目標 | **100 萬次付費乘車/週**（現 50 萬次/週）|
| 近期融資 | $160 億（估值 $1,260 億）|

**擴張計畫**：20+ 座新城市，含首座海外城市**東京**及**倫敦**

## 與 Apple Project Titan 的對比

Apple 耗費數十億美元後放棄，Waymo 以溢價收購——意義在於：
1. **基礎設施稀缺性**：5,500 英畝專用自駕測試場規模難以複製
2. **Waymo 積極擴張**：從開發到部署流程整合，鄰近其亞利桑那整合廠（239,000 sq ft）
3. **Apple 遺產的最終轉移**：一線科技公司的自駕遺產流向目前領先的 Robotaxi 營運商
