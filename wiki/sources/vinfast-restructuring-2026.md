# VinFast 越南業務重組：$5.3 億資產分拆，轉型輕資產模式

**原文**：Bloomberg｜**發布**：2026-05-12（VinFast 公告日）

## 交易結構

| 項目 | 內容 |
|------|------|
| 交易金額 | 約 **US$530M（VND 13.3 兆）** |
| 賣出對象 | VFTP（越南製造子公司）製造資產 |
| 買方 | Future Investment Research and Development JSC 領銜買方群 |
| 還債 | 從收益中償還約 **US$404.8M** P-Note 債務 |
| 完工時程 | **Q3 2026**（待股東及債權人批准）|

## 新架構

**保留（新設 VinFast Vietnam JSC/VFVN）**：
- 全球研發 + 知識產權
- 銷售與售後系統
- 越南、澳洲、德國子公司

**售出（VFTP）**：越南製造廠（但 VinFast 將以成本加成合約繼續使用工廠）

## 海外廠不受影響

北卡羅來納（美國）、印度、印尼工廠**不在此次重組範圍內**。

## 策略意涵

- 從持有資產轉為「持有 IP + 銷售 + R&D」的輕資產模式
- 目標：**2027 年實現獲利**
- 應對壓力：自 2023 年 NASDAQ 上市後持續虧損與現金流緊張
- 模式類比：類似其他資金受壓 EV 新創的資產分拆策略
