# 蔚來李斌：2026 年中國乘用車零售市場恐下滑 15~20%（2026-06-17）

**原文**：Digitimes｜**發布**：2026-06-17

## 摘要

蔚來創辦人**李斌**於重慶汽車論壇示警：中國汽車產業已進入「決賽最殘酷階段」，預估 2026 年乘用車零售銷量年減 **15~20%**，但同時強調純電（BEV）滲透率仍在加速。

## 關鍵數據

| 指標 | 數值 |
|------|------|
| 2026 前 5 月乘用車零售 | 年減近 **兩成** |
| 2026 全年預估 | 年減 **15~20%** |
| 中國乘用車存量 | **3.7 億輛**（市場趨飽和） |
| 2026-05 NEV 滲透率 | **62.9%** |
| 其中 BEV 佔 NEV 比重 | **>67%** |
| 整體市場 BEV 滲透率 | **>42%**（YoY +10pp）|

## 新勢力四家 2026Q1 財報

| 車廠 | 狀況 |
|------|------|
| **零跑** | 交付破 11 萬（最多），毛利率 9.4%，轉虧 |
| **小鵬** | 產品空窗 + AI/自駕研發投入，虧損擴大 |
| **理想** | L 系改款前降價去庫，毛利率從 ~20% 降至 6.1% |
| **蔚來** | 營收翻倍、毛利率創高，但碳酸鋰漲成本，仍虧損 |

## 李斌判斷框架

1. **增量→存量**：市場從高速成長轉向存量競爭
2. **品牌澄清期**：規格同質化後，品牌價值與使用體驗成關鍵
3. **BEV 仍在加速**：以挪威 ~100% NEV 滲透率為趨勢方向
4. **泥濘路上的馬拉松**：核心技術投入 + 營運效率 → 長期生存
