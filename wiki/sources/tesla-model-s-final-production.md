# Tesla Model S / X 正式停產（2026-05）

**原文**：electrive | Sebastian Schaal｜**發布**：2026-05-11

## 核心事件

- **最後一輛 Model S / X** 於 2026 年 5 月初在加州 Fremont 廠下線
- Model S：**生產 14 年**（首輛生產時間追溯至 2012 年）
- Model X：**生產 11 年**
- Fremont 廠原 Model S/X 產線改建為 **Optimus 人形機器人工廠**（目標年產 100 萬台）

## 銷售數據背景

| 指標 | 數值 |
|------|------|
| 2025 年「其他車型」銷量 | **50,850 輛**（含 Model S、X、Cybertruck）|
| 2025 年 Model 3 + Y 銷量 | 160 萬輛 |
| Fremont 廠 Model S/X 設計產能 | 100,000 輛/年 |
| 德州 Cybertruck 產能 | 125,000 輛/年 |

→ 推算 Fremont 高端產線**實際產能利用率極低**

## Signature Edition 紀念款

- 限量 350 輛（Model S 250 輛 + Model X 100 輛）
- 售價 **$159,420**，僅供選定車主購買
- 外觀：石榴紅漆、金色 Tesla 標誌、金色煞車卡鉗、內裝 "Signature" 刻字
- **爭議**：5 月 12 日交車活動在前一個週末突然延期，已付款車主強烈不滿

## 歷史意義

- Model S 是 Tesla 繼 Roadster（基於 Lotus 底盤）後第一款完全自主開發的車型
- 2012 年量產即奪下「年度最佳汽車」，開創純電高端轎車市場
- 停產宣告 Tesla 從「高端先行」到「平價量產 + 機器人」的戰略轉型完成
