# 越南：Ford × Tasco Auto 合作建設 89 座快充站（2026-04-22）

**原文**：electrive.com｜**發布**：2026-04-22

## 核心資訊

| 項目 | 內容 |
|------|------|
| 合作方 | Ford Motor Company + Tasco Auto（越南當地代理商） |
| 形式 | MoU（備忘錄）簽署於河內 |
| 總目標充電站 | **89 座 DC 快充站** |
| Ford 自建 | **34 座**，預定 2026 年第三季完工 |
| Tasco 自建 | **55+ 座**，預定 2026 年底完工 |
| 現有設施 | Ford 已有 16 座 DC + 34 座 AC 充電站（在越南經銷商/服務中心） |
| 潛在設備來源 | Esky Corporation（Tasco 早前協議合作商） |

## 意義

1. **越南充電網絡快速擴張**：Ford 加入本地充電基礎設施建設，顯示外資品牌在東南亞 EV 市場的布局
2. **對比 VinFast**：越南本土 EV 廠 VinFast 已大規模建設充電網，外資充電投入有助拉平競爭條件
3. **東南亞電動化節奏**：越南是東南亞 EV 滲透率較高市場之一，與泰國、印尼同列重點觀察市場
