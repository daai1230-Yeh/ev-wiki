# 小鵬汽車（Xpeng / XPEV）

中國電動車廠，以 800V 高壓快充平台、ADAS 智駕技術為核心競爭力，2026 年已進入全球 60 個國家/地區，積極推進海外市場布局。

## 基本資訊
- **類型**：中國電動乘用車新創
- **股票代碼**：NYSE: XPEV
- **核心技術**：800V/5C 超快充平台、自研 ADAS（XNGP）

## 主要車型
- **G6**：緊湊型電動 SUV，800V 平台，墨西哥售價約 €40,327
- **G9**：中大型電動 SUV，800V 平台，墨西哥售價約 €54,106

## 技術特色
| 項目 | 規格 |
|------|------|
| 充電平台 | 800V 高壓架構 |
| 充電速率 | 最高 5C |
| ADAS | XNGP 自研智駕系統 |

## 海外市場布局（2026）
- **全球版圖**：已進入 60 個國家/地區
- **墨西哥**（2026-04）：G6、G9 正式上市，合作 3 個經銷商集團（墨西哥城、瓜達拉哈拉、蒙特雷），備件中心 1~5 天交付
- **戰略意義**：墨西哥為繞開美國高關稅、進入北美周邊市場的橋頭堡
