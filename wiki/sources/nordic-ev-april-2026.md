# 北歐四國 EV 滲透率：挪威 98.6%，丹麥 81.9%（2026 年 4 月）

**原文**：electrive.com，2026-05-21
**作者**：Daniel Bönnighausen

## 摘要

2026 年 4 月，北歐四國合計每三輛新車中有**兩輛為純電動車（BEV）**，由挪威和丹麥帶動，芬蘭和瑞典仍在發展中。

## 各國 BEV 滲透率（2026 年 4 月）

| 國家 | BEV 占比 | 備注 |
|------|---------|------|
| **挪威** | **98.6%** | 共 11,103 輛；前十名全為純電 |
| **丹麥** | **81.9%** | 一年前為 63.3%；私人購買 **96.3%** |
| **芬蘭** | 48.8% | |
| **瑞典** | 42.0% | |
| **北歐整體** | ~**67%**（兩輛中有一輛以上）| |

## 各國暢銷車型

| 國家 | 第一名 | 說明 |
|------|--------|------|
| 挪威 | **VW ID.4** | 前十全純電 |
| 丹麥 | **Škoda Elroq** | 最高新登記數 |
| 瑞典 | **Volvo EX40** | |
| 芬蘭 | Toyota Yaris Cross（第一）；Škoda Enyaq BEV（第二）| 混合動力仍有競爭力 |

## 市場意義

- 挪威：新車市場**實質完全電動化**（98.6%，幾近百分之百）
- 丹麥：私人購車 96.3% 為 BEV，企業/商業購車拉低整體數字
- 芬蘭、瑞典：ICE 和 HEV 仍在前十榜，反映公共充電基礎設施和補貼政策差距
- 北歐是全球 EV 成熟市場的最佳示範——先行一步 5~10 年，給全球其他市場提供路線圖
