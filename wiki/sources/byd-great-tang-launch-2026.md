# BYD 大唐（Great Tang）電動 SUV 正式上市（2026-06-18）

**原文**：electrive.com｜**發布**：2026-06-18

## 規格

| 項目 | 數值 |
|------|------|
| 車長 | 5.26 m，軸距 3.13 m |
| 座位 | 7 人（2+2+3） |
| 電池 | 105.7 kWh / 130.1 kWh（二代刀片 LFP） |
| 電壓 | **1,000V** 高壓架構 |
| 快充 | **9 分鐘 10→97%**（閃充）|
| 單馬達功率 | 300~370 kW |
| 四驅最大功率 | **585 kW** |
| 最高速度 | 250 km/h |
| 續航 | 800~950 km CLTC |
| 售價 | 239,900~309,900 人民幣（≈€30,800~€39,800）|

## 配備亮點

- 雙腔空氣懸掛（100mm 調節）
- 後軸轉向（最大 7°，轉彎半徑 5.2m）
- 「螃蟹模式」橫向移動
- **天神之眼 5.0** ADAS（車頂 LiDAR 標配）

## 市場動態

- 北京車展（4 月）開放預訂，至今已達 **15 萬筆預訂**
- 王朝系列旗艦，大漢（轎車）稍後跟進
