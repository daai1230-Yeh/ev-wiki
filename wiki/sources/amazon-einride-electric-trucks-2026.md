# Amazon 向 Einride 租賃 75 輛電動卡車部署美國中程物流（2026-04-22）

**原文**：electrive.com｜**發布**：2026-04-22

## 核心資訊

| 項目 | 內容 |
|------|------|
| 買方 | Amazon（美國） |
| 供應方 | Einride（瑞典電動物流方案商） |
| 數量 | 75 輛電動卡車 |
| 模式 | 租賃（Einride 擁有並管理車隊） |
| 應用 | 美國**中程物流**（middle-mile）網絡 |
| 預計里程 | 每年最多 **300 萬英里** |
| 車型 | 非 Einride 自動駕駛 eBot；為**有人駕駛商用 BEV**（製造商未公開） |

## 重點說明

- Amazon **不購買**車輛，而是透過租賃模式降低資本支出
- Einride 提供專屬車隊最佳化軟體及基礎設施服務
- 此次部署的車輛並非 Einride 旗艦無人駕駛 Gen 2 eBot（無駕駛艙/方向盤），而是由其他製造商生產的標準電動卡車

## 意義

1. **電商物流電動化加速**：Amazon 以大規模訂單展示中程物流轉電動的商業可行性
2. **Einride 車隊管理模式**：從賣車轉為提供全套服務（車輛＋軟體＋基礎設施）的「MaaS」模式
3. **美國商用 EV 市場**：補充了 Rivian、Freightliner 等供應商的商用電動生態系
