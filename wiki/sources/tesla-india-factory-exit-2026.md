# Tesla 宣布終止印度設廠計畫

**原文**：多家媒體報導，2026-05-19；Google Alerts 彙整，2026-05-22

## 摘要

印度重工業部長 H.D. Kumaraswamy 於 2026 年 5 月 19 日**公開確認**：Tesla 已決定終止在印度建立電動車製造工廠的投資計畫。印度政府對此表示遺憾，稱 Tesla「不識抬舉」。

## 關鍵事實

- **確認時間**：2026-05-19，由印度重工業部長 H.D. Kumaraswamy 公開說明
- **主要障礙**：關稅問題（Tesla 要求在提高在地化之前先降低進口整車關稅）
- **印度政府回應**：已為 Tesla 量身定制 EV 政策優惠，Tesla 拒絕是不明智之舉
- **背景**：Tesla 2023 年以來多次接觸印度市場，馬斯克曾訪印但最終決定放棄設廠

## 為何重要

| 角度 | 意義 |
|------|------|
| 印度 EV 市場 | 少了 Tesla 品牌拉動效應，中高端市場由 Tata、Mahindra、現代、Kia 和 VinFast 主導 |
| Tesla 全球佈局 | 顯示印度建廠壁壘（關稅、規模效益）過高，Tesla 選擇放棄 |
| 中國品牌機會 | VinFast（已在印度）、BYD（嘗試進入）等可填補 Tesla 撤退的形象空缺 |
| 政策信號 | 印度「先建廠再降稅」政策堅持，可能令其他外資車廠重新評估進入策略 |

## 與 Tesla FSD 印度的關係

Tesla 雖退出印度製造，但**進口銷售**和 FSD/軟體佈局仍可能存在——目前尚未有明確宣告。
