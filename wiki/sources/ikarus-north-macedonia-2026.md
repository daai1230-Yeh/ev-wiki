# Ikarus 贏得北馬其頓 150 輛電動巴士大標（2026-05-05）

**原文**：Electrive（Florian Treiss）｜**發布**：2026-05-05

## 摘要
匈牙利巴士製造商 Ikarus 獲得北馬其頓交通部招標，150 輛電動巴士為 Ikarus 史上單次最大訂單，標誌匈牙利電動巴士廠進入巴爾幹半島市場。

## 訂單概況

| 指標 | 數值 |
|------|------|
| 數量 | **150 輛** Ikarus 120e V4 + **75 座充電站** |
| 規格（最低要求）| 325km 續航 / ≥360kWh 電池 / 充電站 ≥120kW |
| 保固 | 巴士 + 電池 **4 年** |
| 備件供應 | **10 年** |
| 部署 | 斯科普里 ~100 輛 + 其他城市 50 輛 |

## 車型：Ikarus 120e V4

- 長度：12.9 公尺（標準單層城市巴士）
- 電池：LFP，多種容量配置
- 續航：**350–580 公里**（視電池配置而定）

## Ikarus 品牌歷史

- 鼎盛期：曾出口超過 30,000 輛巴士至東德
- 2007 年破產；2010 年復興，近 5 年推出電動巴士
- 現行電動巴士：2 輛在德國圖林根邁寧根 + Chemnitzer Verkehrs-AG 測試中

## 意義

- 巴爾幹半島（東南歐）加入電動巴士版圖
- **歐洲本土（非中資）電動巴士廠的中東歐市場拓展**
- 政府採購模式：整包（巴士 + 充電站 + 保固）
