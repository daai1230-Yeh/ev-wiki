# 駐外新聞：歐洲汽車製造國 2025 年批量摘要

台灣駐德國（法蘭克福+柏林）、斯洛伐克、匈牙利、奧地利、波蘭辦事處 2025 年共發出約 65 份電動車及汽車產業相關電函。核心主軸：**德國汽車業危機與中國 EV 滲透**、**中歐電池供應鏈建設**（斯洛伐克 Gotion、匈牙利 CATL/BYD/EVE）、**ZF 電氣化業務易主**。

---

## 一、德國：汽車供應商危機 + 中國 EV 滲透（2025）

### 1-1 供應商大裁員（2025-01）

德國汽車一階供應商組成勞資聯盟，聯名致函聯邦總理（Bosch + ZF + Continental + Schaeffler + Mahle + IG Metall）：

| 廠商 | 裁員規模 |
|------|------|
| Bosch | 約 **12,000 人** |
| Continental | 約 **12,200 人** |
| ZF | 約 **14,000 人** |
| Schaeffler | 約 **4,700 人** |
| 2019 年以來合計 | 近 **5 萬人** |

根本困境：EV 轉型 + 車用軟體轉型 + 中國競爭 → 德國製造商難降成本，企業出走風險高。

### 1-2 ZF 出售電氣驅動業務（2025-02）

德國第二大汽車零組件商 ZF 計劃分拆出售：
- 出售範圍：**傳統 ICE 變速箱 + PHEV + 純電動技術**（€115 億元營收，占 ZF 總營收 1/4，影響 **3.2 萬名員工**）
- 出售時程：**2026 年**
- **最有力買家：鴻海科技集團**（已自 2023 年起合作，持有 ZF 車軸子公司 50% 股權）
- 其他潛在買家：韓國、德國、美國競爭對手

參見：[[entities/鴻海科技]]

### 1-3 中國 EV 在德國市場滲透（2025-10 報告）

駐德國代表處 2025-10 報告：

| 指標 | 數值 |
|------|------|
| 中國 EV 在德國新車登記占比 | **1.57%**（全車種）|
| 中國 EV 在德國純電市場占比 | **11.5%** |
| BYD 歐洲 H1 2025 銷售 | **9.6 萬輛** |
| 德國 BEV 新車市占率 | **18.4%**（2025 全年）|

德國三大整車廠因應策略：
- **VW**：MEB 模組平台攻大眾化電動車（ID.3/ID.4）
- **BMW**：聚焦高端（i4、iX），強調駕駛體驗 + 數位整合
- **Mercedes**：鎖定豪華電動車（EQS），主打創新舒適

政策：2030 年前建置超過 **100 萬個充電樁**目標。

### 1-4 台灣供應鏈機會（駐德報告 2025-10）

台灣在以下領域對德國汽車轉型具優勢：
- **OTA 軟體更新**與領先 ADAS 感測 + 影像辨識技術
- **半導體製造**與控制晶片設計（台積電為核心）
- **BMS（電池管理系統）**與功率電子製造技術
- **鴻海 EV 模組化平台**（Fisker 代工基礎 → 向德廠提案）

### 1-5 其他德國觀察
- **氫能車德國小型車市場黃金時代落幕**（2025）
- **e-Fuels（合成燃料）市場**：德國重要補充路線，2025 年發展現況報告
- **Nexperia（安世半導體）晶片出口爭議**：中荷衝突衝擊日本汽車業，德國亦面臨供應短缺風險

## 二、斯洛伐克：中歐電動車製造新重鎮

### 2-1 Kia Kosice 工廠：2024 年產量破紀錄

- 韓國起亞汽車斯洛伐克廠 2024 年生產 **351,000 輛**（歷史新高）
- 同年 Kia 全球銷售亦創歷史新高
- Kosice 廠是斯洛伐克最重要的汽車製造據點之一

### 2-2 Volvo Kosice 新廠 + Polestar 7（2025-02 / 09）

Volvo 在斯洛伐克東部 Kosice Valaliky 市興建全新電動車廠：
- **2026 年中**：試產（2027 年初正式量產）
- **第一款車型**：Volvo 全電動小型 SUV
- **第二款車型**：**Polestar 7 SUV**（2028 年起）
- 2025-09 啟動員工招募，首波 **1,000 人**，目標 **3,300+ 人**
- Polestar 7：高端純電 SUV，彌補 Polestar 品牌空白
- Polestar（吉利系）39% 由中國 PSD 投資，24% 吉利控股

參見：[[entities/Volvo]] | [[entities/吉利汽車]]

### 2-3 Gotion InoBat：€12 億電池廠動工（2025-10）

**中國國軒 × 斯洛伐克 InoBat** 合資「GIB EnergyX Slovakia」電池廠：

| 項目 | 內容 |
|------|------|
| 地點 | Nitra 省 Surany 工業園區 |
| 總投資 | **€12 億**（斯洛伐克史上第二大外人投資案）|
| 廠區面積 | 94 公頃 |
| Phase 1 產能 | **20 GWh**（2026-09 試量產，2027 年投產）|
| Phase 2 產能 | 最高 **40 GWh**（可擴增）|
| 就業 | Phase 1：**1,300 人**；Phase 2：**3,500 人** |
| 研發中心 | 50 人規模 |

斯洛伐克政府態度：总理 Fico 出席動工典禮，宣稱為「前瞻性決策」；反對歐盟對中國 EV 徵關稅。
環保疑慮：當地 30 名居民抗議，憂土地與水源影響。

早先 2025-02 斯政府即宣布此案為**戰略性投資**並提供稅收優惠。

參見：[[entities/國軒高科]] | [[concepts/電池技術路線]]

### 2-4 其他斯洛伐克觀察
- 台灣廠商：貿聯控股 + 台達電在斯洛伐克的投資現況（2025-04）
- 美歐關稅戰對斯洛伐克衝擊：出口 GDP 占比極高，高度依賴汽車出口
- 仁寶集團：規劃在**羅馬尼亞**設立車用軟體研發中心
- 台灣鋒明興業（DDK）：在羅馬尼亞 Timisoara 設立組裝據點

## 三、匈牙利：中歐最重要的中國 EV 投資橋頭堡

### 3-1 BMW Debrecen iFactory + 電池供應商（2025-01）

BMW 匈牙利德布勒森（Debrecen）工廠（iFactory）：
- 2025 年啟動 **Neue Klasse** 全電動車生產
- 兩大電池供應商均為中國廠：
  - **CATL**：主供電池
  - **億緯鋰能（EVE Power）**：2025-01 獲環境使用許可，廠址緊鄰 BMW 廠，投資 **4,000 億福林**（~TWD 337 億），採用廢水回收技術

參見：[[entities/寧德時代]] | [[entities/BMW]]

### 3-2 SK On 匈牙利廠裁員（2025-02）

- **韓國 SK On** 匈牙利廠裁員，反映歐洲電動車需求下滑對韓系電池廠的壓力
- 背景：SK On 全球虧損，放緩歐美投資節奏

### 3-3 BYD 匈牙利歐洲總部 + R&D（2025-05）

匈牙利總理 Viktor Orbán 與 BYD 簽署策略合作協議（2025-05-15）：

| 項目 | 內容 |
|------|------|
| 歐洲總部遷移 | 從荷蘭遷至**布達佩斯 Ujbuda 區** |
| 就業 | **2,000 名高階專業人員**（銷售、售後、測試、R&D）|
| R&D 投資 | **€2.5 億**（AI 駕駛技術 €1.05 億 + 動力技術 €1.41 億）|
| 政府補助 | 匈牙利提供 **€5,000 萬**支持 |
| 合作大學 | 至少 3 所匈牙利大學 |
| 專利本地化 | 承諾至少 **50%** 在匈牙利本地註冊 |

匈牙利一貫立場：反對歐盟對中國 EV 關稅，主張合作而非對抗。

### 3-4 BYD Szeged 工廠：延後量產（2025-07）

BYD 歐洲首座電動車廠（匈牙利 Szeged）：

| 項目 | 內容 |
|------|------|
| 投資額 | **€40 億** |
| 計畫最大年產能 | **15~35 萬輛**（乘用車 + PHEV）|
| 原訂量產 | 2025 下半年 |
| 實際延後 | **2026 年**才開始生產，初期僅**數萬輛** |
| 2027 年產量 | 「小幅提升」，仍遠低於產能 |
| 理由 | 土耳其廠建設加速（勞動成本更低），部分產能轉移 |

BYD 另計劃在匈牙利新建**電動卡車廠**（2025-06）。

**意涵**：BYD 雙線布局——匈牙利作為歐盟市場銷售基地 + 土耳其作為低成本製造基地；匈牙利工廠延誤打擊歐盟吸引中國高薪製造業的努力。

## 四、奧地利：供應商壓力 + 鋼材供應

- **Voestalpine（奧鋼聯）**：成為 BYD 匈牙利廠的電動車**鋼材供應商**，奧地利鋼鐵龍頭借中國 EV 歐洲擴張受益
- **KTM 危機**：印度 Bajaj 集團出資 **€6 億**接手瀕臨破產的奧地利重型機車製造商 KTM（驗證印度資金進入歐洲工業的新趨勢）
- **德奧供應商危機**：供應鏈研究所（ASCII）分析歐奧供應鏈過度依賴中國的風險
- **中國稀土管制**：2025 年中國稀土出口管制收緊對奧地利產業的威脅
- **E-Transport 新政**：獎勵充電站 + 電動公車/貨卡車/機車，Kite Rise 新創主推家用鈉離子儲能系統

## 五、波蘭：台波供應鏈合作新進展

- **台塑新智能（FPG New Smart Energy）× 波蘭 SKB 集團** 簽署合作備忘錄（電動車電池相關）
- **成運汽車**：委請駐波蘭辦事處協助拓展電動巴士潛在商機
- **台波智慧移動論壇（2025）**：推動雙邊 EV 產業合作
- EMP（Electro Mobility Poland）修改電動車開發計畫

## 整體意涵

2025 年中東歐電動車供應鏈出現明確的中國主導特徵：
1. **斯洛伐克**：Volvo/Polestar 廠（吉利系）+ 國軒電池廠（€12 億），一國同時承接整車 + 電池製造
2. **匈牙利**：BYD 歐洲總部 + Szeged 整車廠 + CATL/EVE 電池供應鏈（BMW 配套），已成中歐最重要的中國 EV 橋頭堡
3. **德國**：供應商危機（裁員 5 萬+），ZF 賤賣電氣化核心部門（最可能買家：鴻海），中國 EV 已拿下德國純電市場 11.5% 份額

台灣機會：OTA/ADAS/BMS/半導體、鴻海 EMP 波蘭廠、台塑新智能波蘭合作、成運汽車電動巴士。
