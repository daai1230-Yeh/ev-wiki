# Redwood Materials 裁員 10%、轉型儲能業務（2026-04-22）

**原文**：electrive.com｜**發布**：2026-04-22

## 核心資訊

| 項目 | 內容 |
|------|------|
| 裁員數 | **135 人（約 10%）** |
| 此前裁員 | 5 個月前裁減 5% |
| 創辦人 | JB Straubel（特斯拉共同創辦人） |
| 核心業務轉向 | 電池回收 → **+ 電池儲能（BESS）** |
| 北美市場地位 | 電池回收**市場領導者** |
| 年處理量 | **20+ GWh** 電池 |

## 背景

Redwood Materials 成立於 2017 年，聚焦電動車電池回收再利用。近期面臨：
1. 員工規模過快膨脹（Straubel 自承）
2. 電動車電池回收市場成長不如預期
3. 轉向 BESS（電池儲能系統）尋找新增長點

## 意義

- 電池回收行業正面臨**商業化節奏壓力**，廢舊 EV 電池數量仍不足以支撐預期規模
- 儲能轉型反映整個電池產業向 BESS 市場移動的趨勢（詳見 [[concepts/電池技術路線]]）
