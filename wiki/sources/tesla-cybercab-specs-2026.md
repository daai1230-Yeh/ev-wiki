# Tesla Cybercab EPA 正式規格揭露

**事件日期**：2026-06-15｜**文件來源**：美國 EPA 官方申報文件

## 核心規格（EPA 文件確認）

| 規格 | 數值 |
|------|------|
| 重量 | **3,113 磅（~1,412 kg）** — 美國市場最輕 EV |
| GVWR | 3,730 磅（載重空間 617 磅）|
| 驅動 | **前輪驅動（FWD）** — Tesla **首款前驅車** |
| 馬達 | 單前置永磁馬達，**169 馬力（163 kW）** |
| 電池 | **146 Ah × 326V ≈ 48 kWh** |
| EPA 實測 | 實驗室 418.2 英里；× 0.7 係數 ≈ **293 英里（~472 km）實際** |

> Tesla 高管曾表示「接近 300 英里」，與 EPA 預估吻合。

## 量產狀態

- **生產已啟動**（Tesla 官方聲明）
- 原始設計：**無方向盤、無踏板**（真正 Robotaxi 設計），測試版後加裝
- **目標售價**：< $30,000（上市前 2027 年）
- 上市時程：「2026 年前」— 仍待最終確認

## 技術意義

| 項目 | 意義 |
|------|------|
| 1,412 kg 車重 | 比最輕 Model 3 輕約 317 kg；媲美 Honda Civic Touring |
| 48 kWh / FWD | 超輕量化設計取捨：捨棄大電池換極低車重 |
| Tesla 首款 FWD | 顛覆 Tesla 一貫 RWD/AWD 策略 |
| 293 英里 | 都市 Robotaxi 場景足夠；無長途需求 |

## 尚待確認

- **正式上市日期**（宣稱「2026 年前」尚未落地）
- **最終定價**（目標 <$30K 未官方確認）
- **自動駕駛授權範圍**（僅少數城市測試，未全國部署）
