# 歐洲電動卡車重大進展：Daimler eActros Lowliner + Scania 法國廠

**事件日期**：2026-06-04~05｜**地區**：歐洲

## 一、Daimler Truck eActros Lowliner（2026-06-04）

### 產品規格

| 指標 | 數值 |
|------|------|
| 量產時間 | **2027 年** |
| 最大續航 | **500 公里**（滿載） |
| 類型 | 重型純電卡車（BEV）|
| 特點 | **低車架（Lowliner）**設計，利於裝卸作業 |
| 充電 | 支援直流快充 |

### 市場意義

- eActros Lowliner 是 Mercedes-Benz Trucks（Daimler Truck 旗下）向**500公里以上長途重卡**延伸的關鍵機型
- 打破「電動重卡只適合短途配送」的認知盲區
- **競爭者**：Volvo FH Electric（340 公里）、Renault Trucks E-Tech T（500 公里目標）、特斯拉 Semi（840 公里，尚未量產歐洲版）

### 低地板設計的商業優勢

低車架（Lowliner）設計主要服務**超市、物流中心、食品配送**等需頻繁裝卸的場景，降低跳板高度差、加快裝卸速度。

## 二、Scania 法國 Angers 電動卡車廠（2026-06-05）

### 投資概況

| 指標 | 數值 |
|------|------|
| 投資金額 | **€7,000 萬（7,000 萬歐元）**|
| 地點 | 法國 **Angers**（曼恩省）|
| 用途 | 電動卡車生產升級 |
| 母公司 | Scania（隸屬 Traton 集團，VW 旗下）|

### 背景

- Angers 廠是 Scania 在法國的主要生產基地
- 本次投資是 Scania 響應歐盟 2035 年重型車輛 CO₂ 法規的產能電氣化佈局
- Traton 集團（VW 商用車部門）旗下 Scania、MAN、Navistar 均在加速電動化投資
