# 中國收緊鋰電池技術輸出：信實工業轉向直購 CATL 組件（2026-05）

**原文**：Bloomberg / Digitimes｜**發布**：2026-05-21

## 核心事件

中國近期**加強對電池技術轉移的管制態度**，導致印度信實工業（Reliance Industries）無法取得鋰電池生產技術，被迫轉向直接採購中國製成品。

## 信實工業時間線

| 時間 | 事件 |
|------|------|
| 早期 | 與寧德時代（CATL）洽談技術轉移協議 |
| 談判結果 | **破裂**（中方不願轉移核心技術）|
| 轉向 | 改與廈門**海辰儲能**合作，作為主要 BESS 夥伴 |
| 近況 | 與海辰合作不順 → 再次接觸 CATL，但這次**不要技術，只採購產品** |
| 工廠地點 | 印度古吉拉特邦（Gujarat）|

## 中國技術封鎖的更大背景

- 中國正收緊電池技術出口管制（技術轉移紅線）
- 印度 Tata Agratas + JSW Motors 已合計投入 **>$10 億**建立本土電池 R&D（Batch 17 已收錄）
- 阿達尼（Adani）集團董事長 2025 年親赴 CATL 中國總部參觀自動化儲能產線
- 寧德時代在德、匈、西班牙建廠，透過 Ford、Tesla 授權規避美國市場障礙

## 戰略意涵

1. **印度夢想本土化電池生產**，但中國封鎖技術轉移 → 印度仍依賴中國電池成品
2. 信實轉向「直購組件組裝」是務實妥協，但放棄了 IP 自主性
3. CATL 的策略：不輸出技術（保持壁壘），但樂於輸出成品（創造收入）
