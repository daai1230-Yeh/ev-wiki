# 奇瑞 QQ3 上市力抗比亞迪海鷗：平價純電小車市場白熱化

**原文**：DIGITIMES／徐畇融｜**發布**：2026-04-07

## 摘要
奇瑞推出 QQ3，限時售價人民幣 58,900 元（約 $8,500 USD），已累積近 57,000 份預訂，直接挑戰比亞迪海鷗在平價純電市場的地位。

## QQ3 規格

| 項目 | 規格 |
|------|------|
| 電池 | 29.48 kWh / 41.28 kWh（LFP）|
| 動力 | 55 kW / 90 kW，後驅 |
| 續航（CLTC）| 310 km / 420 km |
| 快充（30%→80%）| < 16.5 分鐘 |
| 車身尺寸 | 4,195×1,811×1,573 mm，軸距 2,700 mm |
| 晶片 | Qualcomm Snapdragon 8155 |
| 螢幕 | 8.8 吋儀表 + 12.8 吋多媒體（頂規 15.6 吋 2.5K）|

## 定價
- 限時優惠：人民幣 58,900 元
- 標準售價：人民幣 68,900 元起
- 頂規 420 Shared：人民幣 78,900 元

## vs 比亞迪海鷗
- 海鷗：2026 年 2 月仍為中國出口量最高的 NEV
- QQ3 車身尺寸稍大、軸距更長，後座腿部空間 977mm
