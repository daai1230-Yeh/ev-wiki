# Honda 印度大反攻：6 款新車 + 本地製造 EV（2026-05）

**原文**：ETAuto / Economic Times Auto，2026-05-22

## 摘要

Honda Cars India 宣布 FY2026-27 將推出 **6+ 款新車**，以 ZR-V 混合動力旗艦 SUV 為核心，同時確認本地製造 EV 計畫（Honda 0 Alpha 平台為基礎），目標雙位數成長止住市占流失。

## 2026 年主力車型

| 車型 | 說明 |
|------|------|
| City 改款 | 主力轎車更新 |
| **ZR-V** | 旗艦 SUV，**混合動力獨賣**（強混 2.0L NA），完全建置進口，2026-07 起交車 |
| Elevate 改款 | 緊湊 SUV 更新 |
| Prelude | 轎跑返印（2026 年） |
| **Honda 0 Alpha EV** | 預計 2027 年；**本地製造**（Tapukara 廠，拉賈斯坦邦）|

**ZR-V 關鍵決定**：以強混 2.0L NA 取代 1.5L 渦輪汽油，成為**純混合動力**車型，呼應印度市場對油耗的重視。

## 本地 EV 計畫（Honda 0 Alpha 平台）

| 項目 | 內容 |
|------|------|
| 製造廠 | Tapukara 廠（拉賈斯坦邦）|
| 投資規模 | **₹1,200 億**（~$1.4 億美元）|
| 上市時間 | 預計 **2027 年** |
| 平台 | Honda 0 Alpha（全新全球 EV 平台）|

## 2028 年擴張計畫

- 進入**4m 以下緊湊型**和**中型**兩個細分市場
- 強化本地化，降低進口零件依賴

## 市場背景

Honda Cars India 近年市占持續萎縮，此次為重大戰略轉型：從依賴進口到本地製造 EV，從純汽油到混合動力主打，試圖在 Maruti Suzuki / Tata / Hyundai 的強勢市場中重新站穩。
