# 本田 70 年來首次年虧損：$90 億損失，全面押注油電混動（2026-05）

**原文**：InsideEVs | Iulian Dnistran｜**發布**：2026-05-14

## 核心事件

本田（Honda）公布 FY2026（截至 2026 年 3 月）財報，錄得近 **70 年來首次年度虧損**，原因是豪賭 EV 策略失敗後承認 **$90 億（美元）資產減損**。

## 財務影響

| 項目 | 數值 |
|------|------|
| 年度虧損（FY2026）| 首次年度淨虧損（自上市以來）|
| EV 策略損失 | **~$90 億美元** |
| 主要減損項目 | Honda 0 Series 平台（三款車型全取消）+ 加拿大 EV 工廠 |

## 取消的 EV 計畫

| 項目 | 說明 |
|------|------|
| Honda 0 Series 平台 | 客製化 EV 平台，衍生三款車：0 Series SUV、Saloon、Acura RSX |
| 結果 | **三款全取消**，一輛未交付 |
| 加拿大 EV 工廠 | $110 億加元計畫**無限期暫停** |

同時放棄的長期目標：
- **2030 年 EV 佔銷量 20%** 目標取消
- **2040 年純電 / 燃料電池全面轉型** 目標取消

## 混動轉型策略

| 項目 | 細節 |
|------|------|
| 混動投資（未來 3 年）| **4.4 兆日圓（~$280 億美元）**（用於燃油車 + 混動）|
| EV 未來發展預算 | 僅 **$50 億美元** |
| 新混動平台首款車 | **2028 年**上市 |
| 效率目標 | 「全球最高效動力系統」（比 2023 系統提升 10%+）|
| 首款車型 | 四門 Honda 轎車 + Acura 跨界車（均為 SUV AWD 優先，美國市場）|
| 2030 年目標 | 推出 **15 款**混動車型 |

## 未來展望

- CEO 認為純電未來「不切實際」（"not realistic"）
- 預計 2026 年底恢復獲利（摩托車業務 + 削減成本支撐）
- CEO Toshihiro Mibe：加拿大 EV 工廠即使 EV 需求反彈，地緣政治（美加關稅）也令廠址存疑
