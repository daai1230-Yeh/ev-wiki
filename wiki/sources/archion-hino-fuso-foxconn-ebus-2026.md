# Archion 成軍：日野 + 三菱 Fuso 整併，攜手鴻海（Foxconn）電動巴士（2026-05）

**原文**：Digitimes | 江仁傑｜**發布**：2026-05-19

## 核心事件

日野汽車（Hino）與三菱 Fuso 卡客車（Mitsubishi Fuso）整併後的新公司 **Archion** 正式成軍，公布 2032 年度為終年的中期經營計畫。日本商用車市場自此實質集約為 **Archion 與五十鈴（Isuzu）兩大集團**。

## 財務目標

| 指標 | 2026 年度（預估）| 2032 年度（目標）|
|------|-----------------|-----------------|
| 全球銷量 | **23 萬輛**（+6% YoY）| **28 萬輛**（+3 成）|
| 全球營收 | **¥2.425 兆**（+8% vs 整併前合計）| — |
| 營業利益 | **¥1,100 億**（+52% YoY）| — |
| 營益率 | ~3%（現狀）| **>10%**（目標）|

## 品牌與生產策略

- **繼續保留日野 + 三菱 Fuso 雙品牌**
- 全球總產量 **85%** 以上在共通平台生產
- 東南亞為海外核心市場：6.4 萬 → **8.5 萬輛**（目標）
- 提供柴油 → EV → FCV 完整產品線

## Foxconn 電動巴士合資

| 項目 | 細節 |
|------|------|
| 合資宣布時間 | **2026 年 1 月** |
| 預計成立時間 | **2026 年下半** |
| 生產地點 | 日本富山市（三菱 Fuso 巴士製造工廠）|
| 市場 | 日本研發製造，主要銷往**海外** |
| 現況 | 準備工作按計畫進行（CEO Karl Deppen 確認）|

## 意義

Archion 是 Hino（豐田旗下）+ Daimler Truck 旗下 Mitsubishi Fuso 的歷史性整合。與鴻海（Foxconn）合作電動巴士，代表日本傳統商用車廠透過台灣科技製造商加速電動化布局。
