# 鴻海 CDMS 模式：電動車從電池到整車的垂直整合一條龍

**原文**：DIGITIMES／杜念魯｜**發布**：2026-04-15

## 摘要
鴻海在車用電子展以全場最大攤位展示垂直整合實力，核心亮點為高雄 LFP 電池中心（1.27 GWh，2026 年量產）、CDMS 全球 240 個據點彈性生產模式，以及與三菱 FUSO 正式合約（2026-01 簽訂）。

## 電池自製化（核心主軸）

| 項目 | 內容 |
|------|------|
| 工廠 | 高雄電池中心 |
| 技術 | LFP（磷酸鋰鐵）為主力 |
| 規劃產能 | **1.27 GWh** |
| 量產時間 | **2026 年**正式量產 |
| 優勢 | 電池佔整車成本 >30%，自製可壓低成本並強化供應鏈掌控 |
| 固態電池 | 展示半固態與固態樣品，量產時程未公布 |

## CDMS 全球製造模式

- **CDMS**（委託設計與製造服務）：委託設計 + 委託製造，類似電子代工邏輯延伸至汽車
- 全球布局：**24 個國家**，超過 **240 個**營運據點
- 彈性依客戶需求、稅負結構、成本與物流條件決定生產地點
- 台灣高雄乘用車產線：不限內銷，可視情況支援部分海外市場

## 量產車型陣容（2026）
- **Model B**：乘用
- **Model C（新款）**：已針對美國市場法規調整設計
- **Model D**：多功能休旅
- **Model U（中巴）**：商用，三菱 FUSO 合作基礎車型

## 鴻海 × 三菱 FUSO 最新進度
- **2025 年**：簽署 MOU
- **2026 年 1 月**：正式合約簽訂完成
- 合作形式：合資公司，在日本開發與銷售電動商用車
- 初期車型：Model T + 中巴 Model U；後續可拓展至其他海外市場

## 二輪/三輪
- 目前聚焦電機馬達 + 電池關鍵零組件供應
- 與台灣業者已有電機馬達代工合作
- 電池部分：最快 **2027 年**有成果
