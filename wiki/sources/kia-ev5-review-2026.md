# Kia EV5 試駕：不是最快，但最全面（2026-06-18）

**原文**：electrive.com｜**發布**：2026-06-18

## 規格

| 項目 | 數值 |
|------|------|
| 電池 | 81.4 kWh |
| 續航 | 503 km WLTP（FWD GT-Line） |
| 充電架構 | **400V**（非 800V，不同於 EV6/EV9） |
| 最高 DC 功率 | **150 kW**（10-80%：30 分鐘） |
| AC 充電 | 11kW 標配；22kW 選配（€990） |
| 起售價 | €51,990（GT-Line） |

## 評測重點

- **駕駛體驗**：平順且舒適，高速噪音低，適合家庭/長途用途
- **能耗**：高速公路 22.5 kWh/100km（惡劣天氣），良好條件 20.3，市區低於 17
- **能量回收**：方向盤撥片調節，下次啟動保留設定；i-Pedal 真正單踏板停車
- **後排空間**：1.90m 司機後座仍寬敞

## 市場定位

填補 Kia 電動 SUV 產品線空缺——Sportage 同級電動版本，不以規格見長，以全面均衡勝出。
