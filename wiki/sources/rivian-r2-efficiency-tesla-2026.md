# Rivian R2 Performance EPA 效率比較：與 Tesla Model Y 相當（2026-06-19）

**原文**：InsideEVs｜**發布**：2026-06-19

## EPA 認證數據比較

| 指標 | Rivian R2 Performance | Tesla Model Y Performance |
|------|------|------|
| 續航（EPA）| **330 英里** | 306 英里 |
| 能耗效率 | **32 kWh/100 mi** | 32 kWh/100 mi |
| 車重 | ~5,000 磅（~2,270 kg）| ~4,500 磅（~2,040 kg）|
| 馬力 | 656 hp | ~510 hp |
| 0–60 mph | 3.6 秒 | 3.3 秒 |
| 輪圈 | 21 吋 | 21 吋 |
| 售價 | $57,990 | 類似區間 |

**核心發現**：Rivian R2 Performance 比 Model Y Performance **重 ~230 kg、體型更大**，但達到相同能耗效率（32 kWh/100mi），且續航更長（330 vs 306 英里）。

## 意義

1. **Rivian 電驅技術成熟**：在更重、更方正的車體上達到 Tesla 效率，是工程能力的強力訊號
2. **主流市場攻勢**：R2 定位挑戰 Model Y 所主導的中型 SUV 市場，效率平起平坐移除 Tesla 傳統優勢
3. **EV 效率全面提升**：過去「越重越耗電」的假設被打破，造型選擇不再必然犧牲效率

## 背景

Rivian 於 2026-06-10 開始交付 R2（詳見 [[sources/rivian-r2-deliveries-2026]]）。此篇為 EPA 認證後的效率數據深度比較。
