# 巴西

拉丁美洲最大經濟體，電動巴士市場全球重要節點。聖保羅市擁有拉丁美洲最大電動巴士車隊，2022 年起禁止採購新柴油巴士，並承諾 2040 年全市重型車輛 100% 零排放。

## 市場概況（2026）

- 拉丁美洲電動巴士車隊：**超過 10,000 輛**（巴西聖保羅為最大貢獻城市）
- 聖保羅：拉丁美洲電動巴士最大車隊，全球前五
- 中國品牌（BYD 為主）主導市場供應，本土化製造滿足 BNDES 補貼條件

## 聖保羅電動巴士車隊（2026-06）

| 指標 | 數值 |
|------|------|
| 現有電動巴士規模 | **1,759 輛** |
| 最新採購（2026-06）| **265 輛** BYD（D9W + BC22 刀片電池）|
| BYD 在聖保羅累計 | 約 **550 輛** |
| BYD 在巴西全國累計 | 約 **700 輛** |
| 柴油巴士禁採政策 | **2022 年起生效** |

- BYD 本土化：Campinas 底盤組裝 + Manaus 電池生產，符合 BNDES 在地化補貼要求
- D9W：BYD 18m 大容量電動鉸接巴士；BC22：電動城巴，搭載刀片電池（LFP）
- 265 輛為巴西史上**最大單次重型 EV 採購**
- 參見：[[sources/ev-news-ebus-2026-06-17-30]] | [[entities/比亞迪]]

## COP30 零排放承諾

| 承諾 | 期限 |
|------|------|
| 重型車輛 100% ZEV | **2040 年** |
| 平台 | COP30（2025 年貝倫，聖保羅市為主要旗手）|

- 聖保羅市長 Ricardo Nunes 宣布作為 COP30 氣候承諾的一部分
- 挑戰：要在 2040 年實現 100% ZEV 重型車，充電基礎建設擴建是關鍵瓶頸

## 政策環境

- **BNDES 本地化要求**：國家開發銀行補貼要求一定比例本土生產，推動 BYD 等品牌在巴西建廠
- 柴油禁採政策（2022 起）：全面轉向清潔能源公共運輸
- 電動化補貼：州政府層面（聖保羅州）額外支持
