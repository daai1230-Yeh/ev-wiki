# Mercedes CLA EV 試駕評測（2026-06-17）

**原文**：InsideEVs｜**發布**：2026-06-17

## 規格

| 項目 | 數值 |
|------|------|
| 起售價 | $48,600（測試版 $64,780） |
| 電池 | 85 kWh |
| 最大續航 | 374 英里 EPA |
| 充電架構 | **800V**，最高 320kW DC |
| 10-80% 快充時間 | **22 分鐘** |
| 動力 | 268hp（RWD） |
| 特殊裝置 | 2 速變速箱 |

## 優點

- **駕駛質感**：輕盈靈活，儘管車重 4,530 磅，彎道反應佳
- **續航與充電**：374 英里業界頂尖；800V/320kW 充電極快
- **車道保持**：與 GM Super Cruise 同等水準，免手握

## 缺點

- **Tesla Supercharger 相容性**：在 400V 特斯拉超充站僅能 100kW，差距大
- **車窗控制奇特**：非四門獨立控制，後窗無自動上升
- **AI 語音助理**：問餐廳顯示當前位置而非目的地附近，功能不可靠
- **行車警告頻繁**：每次啟動幾乎都觸發 ADAS 錯誤警告（2~5 分鐘後自清）

## 總評

MMA 平台開啟賓士 EV 新世代。駕駛體驗與充電表現出色，但使用細節仍有打磨空間，AI 助手尚未達標。
