# InsideEVs Critical Materials 每週 EV 摘要（2026-06-20）

**電子報**：InsideEVs Critical Materials（週刊）｜**日期**：2026-06-20｜**作者**：Tim Levin, Rob Stumpf, Kevin Williams

## EV 新車動態

| 車款/品牌 | 重點 |
|------|------|
| Ferrari Luce | 買 EV 可提升其他 Ferrari 車款的分配優先序；以 EV 為稀缺車款的准入籌碼 |
| Mercedes CLA EV | 測試有亮點也有令人抓狂之處 |
| Porsche Taycan 2027 | 加入模擬換檔（曾宣稱此功能會讓 EV「更差」）；電池加大 |
| BMW i3 (Neue Klasse) | 需求強勁提前開放訂購；EPA 440 英里、400 kW DC 充電 |
| Rivian R2 Performance | EPA 330 英里、效率 32 kWh/100mi — 與 Tesla Model Y Performance 相同，但重 ~230 kg（見專題）|
| Rivian 裁員 | 裁減數百人（< 2% 員工）；面臨 Level 3 技術未兌現承諾的集體訴訟 |
| Lucid Cosmos | ~$50,000，首款具大眾市場潛力的 Lucid；歐洲專利申請曝光設計 |

## 電池 / 充電動態

| 事件 | 重點 |
|------|------|
| Electrify America 電池儲能 | 在 DCFC 站部署 BESS，離峰充電、尖峰放電；降低成本並減輕電網壓力 |
| 美國 25 萬個充電點 | L1 + L2 + DCFC 合計突破 250,000 點 |
| GM 矽陽極電池 | 過渡技術橋接至固態電池（詳見另篇）|
| Telo MT1 皮卡 | 800V，106 kWh，400 kW 充電 |
| Honda × QuantumScape | 本田新增固態電池合作夥伴（詳見另篇）|

## 自動駕駛 / Robotaxi

| 事件 | 重點 |
|------|------|
| Waymo 二度召回 | 修正 Robotaxi 可能誤入封閉施工車道的問題；推 Premium 會員制 |
| Tesla Cybercab | 搭載「極小電池」——Tesla 有史以來最輕、最省電（比 Lucid Air 省 30%）|
| Uber Robotaxi | Stellantis × Wayve MOU 擴大合作；Lucid × Nuro 休斯頓服務 2027 年啟動（舊金山灣區今年）|

## 特輯：小米 SU7 Ultra（賽道測試）

- 三電機 AWD，1,500 hp，售價 ~$75,000
- 賽道表現「難以置信地寬容」，適合非專業車手
- 曾破紐柏林紀錄
- 不可能進美國；進入歐洲後料引發市場震撼
