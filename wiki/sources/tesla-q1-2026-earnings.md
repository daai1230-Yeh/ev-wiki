# Tesla Q1 2026 財報：獲利超預期，Capex 大增至 $250 億，Model S/X 停產轉 Optimus

**原文**：CNBC ｜ **發布**：2026-04-22

## 摘要
Tesla Q1 2026 調整後 EPS $0.41 超過預期的 $0.37，但收入略低於預期。關鍵策略轉折：Model S/X 停產，Fremont 工廠轉型 Optimus 人形機器人量產基地；全年資本支出大幅提升至 **$250 億**（原預測 $200 億）。

## 關鍵財務數字

| 指標 | 實際 | 預期 |
|------|------|------|
| EPS（調整後）| **$0.41** | $0.37 |
| 總收入 | **$22.39B**（+16% YoY）| $22.64B |
| 汽車收入 | **$16.2B**（+16% YoY）| — |
| 能源部門收入 | **$2.41B**（-12% YoY）| — |
| 淨利潤 | **$477M**（+17% YoY）| — |
| 汽車毛利率（不含積分）| **19.2%**（全年最高）| — |
| Q1 交車量 | **358,023 輛**（+6% YoY，但低於上季）| — |

## 重大策略決定

### 1. 資本支出大幅提升
- 2026 全年 Capex：**超過 $250 億**（原預測 $200 億，再增 $50 億）
- Q1 Capex 已跳升 67% 至 $24.9 億
- 用途：基礎設施、製造產能、AI/自動駕駛研發

### 2. Model S/X 停產 → Optimus 工廠
- Fremont 廠原生產 Model S/X，將**全面轉型**為 Optimus 人形機器人量產基地
- Q2 開始「首個 Optimus 大型工廠」準備工作
- Optimus 年產量目標：**100 萬台/年**
- Musk 預計「7 月底~8 月」發布下一代 Optimus 機器人

### 3. Hardware 3 用戶無法升級無人監督 FSD
- 搭載 HW3 的 Model 3/Y 舊款**不支援**未來無人監督 FSD
- Tesla 將推出折扣硬體升級換購方案（HW5 升級路徑）
- 影響：大量已購 FSD 的 HW3 車主面臨需要換車或付費升級

### 4. 更平價的 Model 3/Y 版本
- Tesla 確認計畫推出 Model Y 和 Model 3 的「更平價車型」

## 市場背景
- Tesla 股價 YTD 下跌 14%，落後所有大型科技股
- 面臨中國競爭對手（比亞迪、小米）高科技低成本攻勢
- CEO Musk 政治活動引發消費者抵制，尤其歐洲和加州
