# LGES 入股芬蘭正極材料廠：強化歐洲本地電池供應鏈（2026-05）

**原文**：Digitimes | 蔡云瑄｜**發布**：2026-05-18

## 核心交易

樂金能源解決方案（LGES）向芬蘭礦物集團（Finland Mineral Group，FMG）收購 **Espring Finland New Materials 1.7% 股權**。

## Espring 公司背景

| 項目 | 細節 |
|------|------|
| 成立時間 | **2024 年** |
| 股東結構 | FMG **30%** + 中國當升科技 **70%** |
| 工廠地點 | 芬蘭 **科特卡市（Kotka）** |
| 投資額 | **€8 億（~USD 9.48 億）** |
| 動工時間 | **2025 年 3 月** |
| 設計產能 | **6 萬噸/年**（正極材料）|
| 目標投產 | **2027 年** |

## LGES 戰略用途

- LGES 計劃採購芬蘭產正極材料，投入**波蘭弗羅茨瓦夫（Wrocław）工廠**
- 打通「電池材料 → 電池成品」歐洲在地價值鏈
- 弗羅茨瓦夫廠：已量產 EV 電池 + ESS 電池，籌備 **46 系列圓柱形電池**生產
- 確保年產能約 **12 GWh**，已獲歐盟補貼

## 意義

中國在全球正極材料生產高度主導（當升科技持 70%），LGES 以少數股東身份切入確保供應保障，同時響應歐盟「去中國依賴」政策方向。
