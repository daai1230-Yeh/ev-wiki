# 德國電動重卡業者滿意度調查：93% 不願回頭（2026-05）

**原文**：InsideEVs | Iulian Dnistran｜**發布**：2026-05-12
**調查機構**：德國 Öko-Institut（生態研究所）

## 核心數據

| 指標 | 數值 |
|------|------|
| 受訪企業數 | **57 家**運輸公司（至少運營一年）|
| 受訪車輛數 | 合計 **>300 輛**電動重卡 |
| 滿意度 | **93% 滿意或非常滿意** |
| 調查期間 | 2025–2026 冬季 |
| 主要車型 | Mercedes-Benz eActros、eEconic、eArocs |

## 業者主要正評

- 可靠性（reliability）
- 駕駛舒適度
- 較低的運行成本

## 持續挑戰

- **購置成本**（sticker prices）仍偏高
- **電網容量**是部署最大瓶頸
  - 調查場站平均電網容量：**1,115 kW**
  - 已安裝充電功率平均：**629 kW**（40 家樣本）
- 公共重卡充電站：使用比例 **< 5%**（場地空間不足是主因）
- 低電費 + 免公路費（road toll exemption）是繼續擴大電動重卡規模的關鍵前提

## 可靠性評估

| 評語 | 比例 |
|------|------|
| 故障率與柴油車相同 | 32% |
| 故障率顯著低於柴油車 | 16% |
| 故障率高於柴油車 | 18% |
| 故障率顯著高於柴油車 | 7% |

## 使用強度

- 日均行駛：**268 英里（432 km）**
- 約 **40% 車隊**至少 40% 車輛日行 >310 英里（500 km）

## 前景展望

- **93% 早期採用者**預期電動重卡將在 **2030 年前**成為公司標準車型
