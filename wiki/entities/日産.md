# 日産（Nissan）

日本第三大汽車製造商（僅次於豐田、本田），雷諾日産三菱聯盟（RNM Alliance）成員。近年深陷財務危機，面臨工廠關閉、裁員及與本田的合併協商破裂後轉為 SDV 技術合作。固態電池研發走在業界前沿，但量產時程延後至 2028 財年。

## 基本資訊

| 項目 | 細節 |
|------|------|
| 總部 | 日本橫濱 |
| 聯盟 | **RNM Alliance**（Renault + Nissan + Mitsubishi）|
| 執行長 | **Ivan Espinosa**（2025 年接任，前品牌長）|
| 年銷量（FY2024）| ~340 萬輛（跌幅顯著）|
| 財務狀態 | 多季虧損，FY2025 Q1 虧損達數千億日元 |

## 本田×日産合併談判：破局轉合作（2025-2026）

### 合併談判（2024-2025）

- 2024 年 12 月：本田、日産宣布探討合併可能
- 2025 年 2 月：**談判破裂**（本田要求實際控制權，日産拒絕）
- 根本矛盾：
  - 本田希望主導合併後集團
  - 日産股東（含雷諾）不願讓步
  - 三菱等聯盟成員立場複雜

### SDV 技術合作（2025-2026）

合併失敗後，雙方轉為技術合作：
| 合作項目 | 細節 |
|---------|------|
| **SDV（軟體定義汽車）** | 與本田、三菱共同開發 SDV 架構 |
| **ECU 共通化** | 整合電子控制單元，降低開發成本 |
| **協議時間** | ~2026 年 7 月確認 |
| **量產時程** | 最早 **2029-2030 年**車型搭載 |

- 三方共通化 ECU 可節省數千億日元軟體開發成本

## 固態電池：23 層驗證，FY2028 量産（橫濱）

日産是全球固態電池研發進度最前沿的傳統車廠之一：
| 項目 | 細節 |
|------|------|
| 驗證進度 | **23 層堆疊固態電芯**驗證完成 |
| 製造技術 | 層疊（Lamination）固態電解質工藝 |
| 目標量産 | **FY2028**（2028 年 4 月～2029 年 3 月）|
| 量産廠房 | **橫濱工廠**（神奈川縣）|
| 目標能量密度 | > 400 Wh/kg（達傳統 LIB 兩倍以上）|
| 目標售價 | 接近傳統鋰離子電池成本 |

- 日産固態電池路線：**全固態**（All-Solid-State）而非半固態過渡路線
- 電解質材料：硫化物系固態電解質（業界最主流方向）
- 挑戰：大面積均勻製造、低溫性能、界面阻抗管理
- 若 FY2028 量產成功，日産將成為全球首家規模量產全固態電池乘用車的廠商

## 英國桑德蘭（Sunderland）廠：奇瑞合作

| 項目 | 細節 |
|------|------|
| 廠房 | 英國 Sunderland 廠（英國最大汽車廠）|
| 合作方 | **奇瑞汽車（Chery）** |
| 合作性質 | 由奇瑞利用日産 Sunderland 廠產能生產奇瑞品牌車型 |
| 動機 | 規避英國/歐盟對中國進口車的高關稅 |
| 日産獲益 | 廠房使用費 + 維持 Sunderland 就業（政治壓力）|

- 英國政府給予關稅豁免前提：在英國本地組裝（非整車進口）

## 美國密西西比州 EV 廠：放棄計畫

- 原計劃在密西西比州興建電動車生產線（與本田合作架構下）
- 隨 Honda-Nissan 合併破局，計畫**正式放棄**
- 影響：美國本土 EV 生產計畫大幅縮水；密西西比州失去預期投資

## 稀土依賴減少：-90% 計畫

| 項目 | 細節 |
|------|------|
| 目標 | 電動馬達稀土使用量減少 **90%** |
| 技術路線 | 鐵素體磁鐵（Ferrite Magnet）+ 拓撲優化設計 |
| 時程 | 2026-2028 年新型電動馬達 |

- 背景：中國稀土出口管制風險使 OEM 加速稀土替代研究
- 日産目標最激進（-90%），若成功可大幅降低對中國稀土的依賴

## 財務危機與重組

- FY2024 淨虧損：**¥749.7 億**（$4.8 億美元）
- 全球裁員：**9,000 人**（約 6% 員工）
- 工廠關閉：全球縮減至 17 座（從 20 座）
- 中國市場：與東風汽車合資廠持續虧損，減持合資比例
